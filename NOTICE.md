# LedCover Notice

LedCover is an independently maintained project derived from earlier open
source cover-generation projects.

## Upstream Projects

- Gudong Cover: https://github.com/maoruibin/GudongCover
- RongCover: https://github.com/zjr3193/RongCover

## Attribution

- Gudong developed and open-sourced the original Gudong Cover project.
- ruibin3 contributed to the original project history.
- Rong Shufu / zjr3193 maintained the RongCover version and added secondary
  editing and Docker deployment improvements.
- LabDogX maintains LedCover as a self-hosted dual-platform cover editor for
  WeChat Official Account and Xiaohongshu covers.

## Maintenance Direction

LedCover keeps attribution to upstream authors while continuing as a clean
independent project. The project focuses on:

- self-hosted deployment
- WeChat Official Account cover generation
- Xiaohongshu cover generation
- reusable templates and color presets
- local and remote AI provider support
- browser-only PNG export
